# 主页布局设计 — 暖 Warmart

> 高端精致风格的 C 端商城主页，全深色暖棕渐变调性。

## 整体信息

| 项目 | 内容 |
|------|------|
| 品牌 | 暖 Warmart |
| 风格 | 高端精致（Premium） |
| 色调 | 全深色暖棕渐变 |
| 布局流 | 杂志式节奏：Hero → 品牌故事 → 活动横幅 → 商品浏览 → 页脚 |

## 配色体系

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景渐变 | `linear-gradient(145deg, #1a1512, #2d231d, #1c1815)` | 暖棕渐变基底 |
| 卡片/组件底 | `#1a1512` | 商品卡、组件容器 |
| 占位/装饰底 | `#2d231d` | 图片占位、次要容器 |
| 主文字色 | `#d4c9b8` | 标题/正文 |
| 辅助文字色 | `#8a7e6b` / `#a89b88` | 标签、描述 |
| 品牌色 (Accent) | `#c0664a` | Tab 选中、价格、小标题 |
| 边框/分割线 | `#3a3028` | 常规边框 |
| CTA 边框 | `#d4c9b8` | 按钮描边 |

## 页面结构

```
┌─────────────────────────────────────┐
│          HeaderNav (导航栏)           │ 固定顶部，滚动后毛玻璃
├─────────────────────────────────────┤
│          全屏 Hero 区域               │ 深色背景 + 文案 + CTA
├─────────────────────────────────────┤
│           品牌故事区域                 │ 左文右数，数据指标
├─────────────────────────────────────┤
│           活动横幅区域                 │ 左文案右图，可轮播
├─────────────────────────────────────┤
│          商品浏览区                    │ 分类Tab + 商品网格
├─────────────────────────────────────┤
│             页脚                      │ 四列链接 + 版权
└─────────────────────────────────────┘
```

## 逐区域设计

### 1. HeaderNav（全局导航栏）

- 固定在视口顶部，z-index 高于所有内容
- 初始状态：透明背景（与 Hero 融合）
- 滚动后：毛玻璃效果（`backdrop-filter: blur(20px)`）+ 深色半透明底
- Logo + 品牌名 "WARMART · 暖" 居左
- 导航链接（首页、商品）居中
- 搜索图标 / 购物车图标（数量角标）/ 用户头像 居右
- 未登录：显示登录/注册入口；已登录：头像下拉菜单
- 响应式：768px 以下隐藏导航文字，保留图标

### 2. 全屏 Hero

- 高度：100vh，暖棕渐变背景
- 品牌名 "WARMART · 暖" 置于顶部区域（小字，letter-spacing）
- 居中文案：
  - 主标语（serif 字体，~36px，细字重）"探寻生活中的温暖质感"
  - 副描述（sans-serif，~14px，辅助色）
- 单个 CTA 按钮："探索新品"（细边框描边，无填充，圆角 2px）
- 背景可配大幅产品摄影或抽象质感图（半透明叠加）
- 底部居中：微弱的向下滚动提示（小圆点或箭头动画）

### 3. 品牌故事

- 暖棕渐变背景，内边距 80px 上下
- 分左右两栏：
  - **左栏**：小标签 "我们的理念" → 标题 "每一件商品都经过精心甄选" → 描述文案
  - **右栏**：2×2 数据网格（精选品牌 200+ / 优质商品 5000+ / 用户好评 98% / 信任用户 10万+）
- 数值用品牌色 `#c0664a`，标签用辅助色
- 进入视口时触发淡入上升动画

### 4. 活动横幅

- 中等高度（~300px），暖棕渐变背景
- 左右布局：文案 + CTA（左） | 活动主图（右）
- 顶部小标签 "限时活动" / "新品上市"
- 支持轮播：底部圆点指示器，左右箭头切换
- 每次展示一个活动，可横向滑动切换

### 5. 商品浏览区

- 暖棕渐变背景，内边距 60px 上下
- **标题区**：小标签 "SHOP NOW" → 主标题 "选购好物"
- **分类 Tab 栏**：横向排列，选中项 `#c0664a` 边框高亮，未选中 `#3a3028` 暗边框。默认选中"全部推荐"
- **商品网格**：4 列（`grid-template-columns: repeat(4, 1fr)`）
  - 商品卡：方形图区域（`aspect-ratio: 1`，`#2d231d` 占位底） → 商品名 → 价格（品牌色）
  - 卡片背景 `#1a1512`，圆角 8px
  - 悬停效果：轻微上浮阴影
- **底部**："查看更多" 按钮（暗边框描边），点击跳转 `/product/list`

#### 响应式断点

| 断点 | 列数 | 调整 |
|------|------|------|
| ≥1024px | 4列 | 默认布局 |
| 768-1023px | 3列 | 缩小内边距 |
| <768px | 2列 | 隐藏 Tab 为下拉筛选 |

### 6. 页脚

- 暖棕渐变背景，内边距 48px
- **上栏**：四列链接布局
  - 品牌介绍（2fr）：品牌名 + 简短描述
  - 购物指南（1fr）：如何下单 / 支付方式 / 配送说明
  - 售后服务（1fr）：退换政策 / 退款说明 / 联系客服
  - 关于我们（1fr）：品牌故事 / 加入我们 / 隐私政策
- 列标题用品牌色 `#c0664a`（小字，letter-spacing）
- **分割线**：`linear-gradient(to right, transparent, #3a3028, transparent)`
- **底栏**：居中的版权信息 "© 2026 Warmart. All rights reserved."

## 动画与交互

| 元素 | 效果 |
|------|------|
| Hero CTA | hover 时填充品牌色背景 |
| 品牌故事 | 滚动进入视口时 fade-in-up |
| 商品卡片 | hover 时 y 轴偏移 -4px + 阴影增强 |
| 分类 Tab | 点击切换，active 态边框色变暖 |
| 导航栏 | 滚动 >80px 触发毛玻璃 |
| 横幅轮播 | 左右滑动切换，底部圆点指示 |

## 技术实现

- 每个区域为独立 Vue 组件，放在 `src/views/home/` 下
- 使用 IntersectionObserver 实现滚动动画
- CSS 变量体系统一管理配色
- 响应式设计，3 个断点：1024px / 768px / 640px
